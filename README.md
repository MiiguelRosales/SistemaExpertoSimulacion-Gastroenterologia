# Sistema Experto de Gastroenterología

Este proyecto es un simulador de un sistema experto para el diagnóstico o evaluación de casos de gastroenterología.

## Autores

- Jahiro Gonzalez
- Miguel Rosales

## Archivos principales

- `Gastroenterologia.pl` - Programa en Prolog que contiene la lógica del experto.

## Uso

1. Abrir el archivo `Gastroenterologia.pl` en un entorno Prolog.
2. Consultar el programa con el predicado correspondiente.
3. Ingresar los síntomas o datos solicitados por el simulador.

## Ejecución web

1. Instala dependencias con `pip install -r requirements.txt`.
2. Ejecuta `python app.py`.
3. Abre el navegador en `http://127.0.0.1:5000`.

## Notas

- Este simulador fue desarrollado como parte de la materia de Programación Lógica y Funcional.
- Ajusta los predicados y reglas según sea necesario para tu caso de uso o para extenderlo con más conocimiento médico.
